Outbreak simulation

Welcome to outbreak simulator

You can try to simulate outbreak with your own parameters. Current default parameters are estimated to be somehow close to covid-19 for naive population, but you can simulate other diseases if you set parameters right . You can change all the parameters live during the simulation except of population size. If you change population size it will restart the simulation with actual chosen parameters. For more accurate simulation with higher population size you can choose to import new infected person every few days during first weeks. There are many improvements to be done. I will be glad to implement good ideas or recommendations if you contact me at ondrej.housar@gmail.com.
